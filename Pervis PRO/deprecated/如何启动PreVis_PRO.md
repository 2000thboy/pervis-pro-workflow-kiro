# 🚀 如何启动 PreVis PRO

## ⭐ 推荐方式：GUI启动器

### 🎯 一键启动（最简单）
```
双击 → PreVis_PRO_启动器.pyw
```

这是**秋叶风格的GUI启动器**，具有：
- 🌙 优雅的深色界面
- 🚫 无CMD窗口弹出
- ⚡ 一键启动所有服务
- 📊 实时状态监控
- 🎛️ 简单易用的操作

### 🖥️ 启动后访问
启动成功后，GUI会自动打开浏览器，或手动访问：

- **🎬 MVP工作流**: http://localhost:3000/mvp-workflow
- **🏠 主界面**: http://localhost:3000
- **📚 API文档**: http://localhost:8000/docs

## 🔧 备用方式

### 命令行启动（开发者）
```bash
python tools/一键启动PreVis_PRO.py
```

### 快速访问工具
```bash
python tools/快速访问.py
```

### 系统诊断
```bash
python tools/系统状态检查.py
```

## 🎯 MVP AutoCut 功能

启动后直接访问：**http://localhost:3000/mvp-workflow**

这是完整的智能自动剪辑工作流：
1. **剧本输入** - 输入剧本内容
2. **Beat分析** - AI自动分析故事节拍
3. **素材搜索** - 智能匹配视频素材
4. **时间轴编辑** - 自动生成剪辑时间轴
5. **预览播放** - 实时预览编辑效果
6. **导出分享** - 渲染输出MP4视频

## 🛠️ 故障排除

### 启动失败
1. 检查Python和Node.js是否安装
2. 运行系统诊断：`python tools/系统状态检查.py`
3. 查看GUI启动器的日志窗口

### 端口占用
- 后端端口：8000
- 前端端口：3000
- 如有冲突，关闭占用端口的程序

### 服务无响应
1. 使用GUI启动器的"停止服务"按钮
2. 重新点击"一键启动"
3. 或使用备用启动器重启
