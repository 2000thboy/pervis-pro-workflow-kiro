# PreVis PRO 简化使用说明

## 🚀 快速启动（推荐方式）

### GUI启动器（秋叶风格，无CMD弹窗）
1. **双击** `PreVis_PRO_启动器.pyw` 文件
2. 点击「一键启动」按钮
3. 等待服务启动完成（首次启动会自动安装依赖）
4. 点击「打开界面」访问系统

### 其他启动方式
- `启动PreVis_PRO.bat` - 批处理启动
- `一键启动PreVis_PRO.py` - 命令行启动

## 📍 访问地址

| 服务 | 地址 |
|------|------|
| 前端界面 | http://localhost:3000 |
| API文档 | http://localhost:8000/docs |
| 健康检查 | http://localhost:8000/health |

## 📁 素材管理

### 素材位置
- **本地素材**: `assets/originals/`
- **网络素材**: `L:\PreVis_Assets\originals\`（如已配置）

### 添加素材
1. 将视频/图片文件复制到素材目录
2. 系统会自动处理新素材
3. 原始文件保持不变

## 🔧 配置工具
- `配置网络盘符.py` - 图形化配置L盘素材库
- `系统状态检查.py` - 检查系统运行状态

## ❓ 常见问题

### Q: 首次启动很慢？
A: 首次启动会自动安装前端依赖（npm install），请耐心等待2-5分钟。

### Q: 启动时报错？
A: 请确保已安装 Node.js 和 Python，然后手动运行：
```bash
cd frontend
npm install
```

### Q: 如何停止服务？
A: 在GUI启动器中点击「停止服务」按钮，或关闭启动器窗口。
