# 🚀 PreVis PRO 团队部署快速指南

## 给测试人员的简单说明

### 📋 测试前准备

#### 服务器端（你的电脑）
1. 确保安装了 Python 3.10+ 和 Node.js 18+
2. 双击运行 `启动PreVis_PRO.bat`
3. 等待两个命令行窗口出现并显示服务已启动
4. 记下你的IP地址（运行 `ipconfig` 查看）

#### 测试人员端
只需要一个浏览器（Chrome/Edge/Firefox）

---

## 🌐 局域网访问设置

### 步骤1：获取服务器IP
在服务器电脑上运行：
```
ipconfig
```
找到 `IPv4 地址`，例如：`192.168.1.100`

### 步骤2：开放防火墙
以管理员身份运行PowerShell：
```powershell
netsh advfirewall firewall add rule name="PreVis Backend" dir=in action=allow protocol=TCP localport=8000
netsh advfirewall firewall add rule name="PreVis Frontend" dir=in action=allow protocol=TCP localport=3000
```

### 步骤3：配置前端API地址
编辑 `frontend/.env` 文件，添加：
```
VITE_API_URL=http://192.168.1.100:8000
```
（将IP替换为你的实际IP）

### 步骤4：重启服务
关闭之前的服务窗口，重新运行 `启动PreVis_PRO.bat`

### 步骤5：分享访问地址
告诉测试人员访问：
```
http://192.168.1.100:3000/mvp-workflow
```

## 📝 测试人员操作指南

### 访问系统
1. 打开浏览器
2. 输入地址：`http://[服务器IP]:3000/mvp-workflow`
3. 看到PreVis PRO界面即成功

### 测试流程
1. **输入剧本** → 在文本框输入任意剧本内容
2. **分析剧本** → 点击"分析剧本"按钮
3. **查看Beat** → 系统自动生成故事节拍
4. **搜索素材** → 输入关键词搜索
5. **编辑时间轴** → 拖拽素材到时间轴
6. **预览播放** → 点击播放按钮预览
7. **导出视频** → 点击"开始渲染"

### 测试剧本示例
```
FADE IN:

EXT. 城市街道 - 白天

繁忙的都市街道，人来人往。

一个年轻人匆忙走过。

CUT TO:

INT. 办公室 - 白天

现代化的办公室。

FADE OUT.
```

## ⚠️ 常见问题

### 无法访问系统
- 检查服务器是否启动（两个命令行窗口）
- 检查IP地址是否正确
- 检查防火墙是否开放端口

### 页面显示错误
- 刷新页面（F5）
- 清除浏览器缓存（Ctrl+Shift+Delete）

### 功能无响应
- 检查后端服务是否正常（访问 http://[IP]:8000/api/health）
- 查看后端命令行窗口是否有错误信息
