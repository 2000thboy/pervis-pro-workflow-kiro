# PreVis PRO 使用指南

## 🚀 快速启动

### 方法1: 一键启动（推荐）
**双击运行**: `启动PreVis_PRO.bat`
**或命令行**: 
```bash
python 一键启动PreVis_PRO.py
```
- ✅ 完全自动化，无需手动操作
- ✅ 自动配置网络盘符
- ✅ 自动启动前后端服务
- ✅ 自动打开Web界面

### 方法2: 快速访问
**双击运行**: `快速访问.bat`
**或命令行**: 
```bash
python 快速访问.py
```
- 🔗 快速打开各种界面
- 📊 检查系统状态
- 💡 提供使用指导

## 🔧 配置工具

### 网络盘符配置
```bash
python 直接配置网络盘符.py        # 命令行配置
python 配置网络盘符.py            # 图形化配置
```

### 系统状态检查
```bash
python 系统状态检查.py
```

## 🌐 访问地址

- **前端界面**: http://localhost:3001
- **API文档**: http://localhost:8000/docs
- **健康检查**: http://localhost:8000/health

## 📁 素材库

- **位置**: L:\PreVis_Assets\originals
- **说明**: 原始文件不会被修改
- **处理**: 系统自动生成代理文件和缩略图

## 💡 常见问题

### 前端无法访问
```bash
python 智能启动器.py
```

### 后端响应慢
- 系统正在处理大量素材文件
- 请耐心等待5-10分钟

### 重新配置网络盘符
```bash
python 直接配置网络盘符.py
```
