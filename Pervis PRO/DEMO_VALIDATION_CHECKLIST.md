## 🚨 常见问题快速修复

### 问题1: Sanity Check失败
**症状**: 脚本显示 ❌ FAIL 状态
**快速修复**:
```bash
# 重启后端服务
cd backend && python main.py

# 重启前端服务  
cd frontend && npm run dev

# 重新运行检查
python sanity_check.py
```

### 问题2: 演示项目加载失败
**症状**: 项目无法加载或显示不完整
**快速修复**:
```bash
# 检查项目文件完整性
ls -la demo_projects/cyberpunk_trailer/

# 验证JSON文件格式
python -m json.tool demo_projects/cyberpunk_trailer/project.json

# 重新加载项目
```

### 问题3: 素材推荐为空
**症状**: Beat分析正常但无推荐结果
**快速修复**:
```bash
# 检查素材数据库
sqlite3 backend/pervis_director.db ".tables"

# 重新生成向量数据
python backend/scripts/rebuild_vectors.py

# 验证推荐功能
```

### 问题4: 预览播放失败
**症状**: 点击素材无法播放
**快速修复**:
```bash
# 检查代理文件
ls backend/assets/proxies/

# 验证FFmpeg可用性
ffmpeg -version

# 重新生成代理文件
python backend/scripts/regenerate_proxies.py
```
