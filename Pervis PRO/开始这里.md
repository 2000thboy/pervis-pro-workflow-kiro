# 🚀 Pervis PRO - 开始这里

## 欢迎使用 Pervis PRO！

这是一个 AI 驱动的智能视频编辑和预览系统。在开始之前，需要安装一些基础环境。

## ⚡ 快速开始

### 方案 A：自动安装（推荐）

1. **以管理员身份运行 PowerShell**
   - 右键点击 PowerShell 图标
   - 选择"以管理员身份运行"

2. **执行安装脚本**
   ```powershell
   cd "E:\Pervis PRO\Pervis PRO"
   .\完全自动安装.ps1 -Force
   ```

3. **等待安装完成**（约 10-20 分钟）

### 方案 B：手动安装

如果自动安装失败，请按以下步骤手动安装：

#### 步骤 1：安装基础环境

1. **Python 3.10+**
   - 下载：https://www.python.org/downloads/
   - ⚠️ 重要：安装时勾选 "Add Python to PATH"

2. **Node.js 18+**
   - 下载：https://nodejs.org/
   - 选择 LTS 版本

3. **Git**
   - 下载：https://git-scm.com/

4. **FFmpeg**（可选，用于视频处理）
   - 下载：https://www.ffmpeg.org/download.html

#### 步骤 2：安装项目依赖

安装完基础环境后，**重启命令行**，然后运行：

```bash
# 双击运行
simple_install.bat
```

或手动执行：

```bash
# 后端依赖
cd backend
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt

# 前端依赖
cd ..\frontend
npm install

# 启动器依赖
cd ..
pip install customtkinter pillow
```

## 🔑 配置 API 密钥

1. **获取 Gemini API 密钥**
   - 访问：https://makersuite.google.com/app/apikey
   - 创建新的 API 密钥

2. **编辑配置文件**
   - 打开 `backend\.env` 文件
   - 将 `your_gemini_api_key_here` 替换为你的实际密钥

```env
GEMINI_API_KEY=你的实际密钥
```

## 🎬 启动项目

配置完成后，运行：

```bash
python 启动_Pervis_PRO.py
```

或手动启动：

```bash
# 终端 1 - 后端
cd backend
python -m uvicorn main:app --reload --port 8000

# 终端 2 - 前端
cd frontend
npm run dev
```

## ❓ 遇到问题？

### 检查环境
```powershell
python --version
node --version
git --version
ffmpeg -version
```

### 常见问题

**问题：命令未找到**
- 解决：确保安装时勾选了 "Add to PATH"
- 重启命令行

**问题：权限错误**
- 解决：以管理员身份运行

**问题：网络超时**
- 解决：检查防火墙设置
- 使用国内镜像：
  ```bash
  pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
  npm config set registry https://registry.npmmirror.com/
  ```

## 🎯 系统要求

- **操作系统**：Windows 10/11
- **内存**：至少 8GB RAM
- **磁盘空间**：至少 5GB 可用空间
- **网络**：需要互联网连接（用于 AI 功能）
