## 🖼️ 第二步：打开文件

### 方法1: 使用文件管理器（推荐）

1. 按 `Win + E` 打开文件管理器
2. 导航到: `F:\100KIRO project\Pervis PRO\exports`
3. 双击文件即可打开：
   - DOCX文件会用Microsoft Word打开
   - PNG文件会用默认图片查看器打开

### 方法2: 使用命令行

打开PowerShell或CMD，运行：

```powershell
# 打开DOCX文件
start exports\demo_cyberpunk_trailer_script.docx

# 打开PNG文件
start exports\demo_cyberpunk_trailer_beatboard.png

# 或者打开整个文件夹
start exports
```

## 💡 提示和技巧

### 查看DOCX文件

1. **编辑模式**: 在Word中可以直接编辑内容
2. **打印预览**: 查看 → 打印预览
3. **导出PDF**: 文件 → 另存为 → PDF

### 查看PNG文件

1. **缩放**: 使用鼠标滚轮或Ctrl + 滚轮
2. **全屏**: 按F11（在大多数图片查看器中）
3. **打印**: Ctrl + P

### 重新生成文件

如果您想重新生成文件：

```powershell
# 删除旧文件
Remove-Item exports\* -Force

# 重新运行演示
python mvp_export_tag_demo.py
```

## 📞 遇到问题？

### 常见问题

**Q: 找不到exports文件夹？**
A: 运行 `python mvp_export_tag_demo.py` 会自动创建

**Q: DOCX文件打不开？**
A: 确保安装了Microsoft Word或WPS Office

**Q: PNG图片显示不正常？**
A: 尝试用不同的图片查看器打开

**Q: 想要PDF格式？**
A: 在Word中打开DOCX，然后另存为PDF
