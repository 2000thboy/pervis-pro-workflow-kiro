## 🚀 推荐启动流程

### 方案 A：完全自动（推荐）

1. **以管理员身份运行 PowerShell**
2. **执行自动安装**：
   ```powershell
   .\完全自动安装.ps1 -Force
   ```
3. **等待安装完成**（10-20分钟）
4. **配置 API 密钥**
5. **启动项目**：`python 启动_Pervis_PRO.py`

### 方案 B：快速检查启动

1. **双击运行**：`检查并启动.bat`
2. **根据提示完成缺失的安装**
3. **配置 API 密钥**
4. **重新运行检查启动脚本**

### 方案 C：手动安装

1. **查看**：`开始这里.md`
2. **按步骤手动安装基础环境**
3. **运行**：`simple_install.bat`
4. **配置并启动**

## ⚙️ 配置要求

### API 密钥配置
编辑 `backend/.env` 文件：
```env
GEMINI_API_KEY=你的实际密钥
LLM_PROVIDER=gemini
```

### 获取 API 密钥
访问：https://makersuite.google.com/app/apikey

## 🎬 启动方式

### 使用启动器（推荐）
```bash
python 启动_Pervis_PRO.py
```

### 手动启动
```bash
# 终端 1 - 后端
cd backend
python -m uvicorn main:app --reload --port 8000

# 终端 2 - 前端  
cd frontend
npm run dev
```

4. **依赖冲突** - 使用虚拟环境隔离
