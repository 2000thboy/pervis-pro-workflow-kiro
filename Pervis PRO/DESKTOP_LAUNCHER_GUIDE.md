## 🚀 快速开始

### 1. 一键启动（推荐）
```bash
# 自动启动后端服务和桌面启动器
python start_pervis.py
```

### 2. 手动启动
```bash
# 先启动后端服务
cd backend
python main.py

# 再启动桌面启动器（新终端）
python pervis_desktop_launcher.py
```

- **桌面启动器**: 原生界面，响应更快
